# Приложение «Happy Birthday»

**Разработчики:**  
Максумов А.М. (группа ИП-23-3)  
Синицын Р.В. (группа ИП-23-3)

## Описание проекта
На главном экране отображается поздравление с днём рождения и подпись. Текст наложен на фоновое изображение с регулировкой прозрачности для лучшей читаемости.

## Функциональность
Основной экран содержит:
- Поздравительный текст (сообщение)
- Подпись отправителя
- Фоновое изображение
